# Plan: `switch --create` flag

## Goal
Add `--create` boolean flag to `switch` command. When set:
1. Prompt for repository (fuzzy picker, or use `--repository` flag)
2. Prompt for new branch name via **text input** (readline question)
3. Create git worktree (`WorkTreeService.create`)
4. Switch tmux to it

## Files to modify

### `src/Sandbox/Cli.ts`
- Add `create` flag to `switchCommand`:
  ```ts
  create: Flag.boolean("create").pipe(Flag.withDescription("Create new branch and switch")),
  ```
- Pass `create` in `service.run(input)`
- Add `GitServiceImpl` to `switchLayer`:
  ```ts
  const switchLayer = SwitchServiceImpl.pipe(
    Layer.provide(Layer.succeedServices(ServiceMap.mergeAll(TmuxServiceImpl, TerminalPicker, GitServiceImpl)))
  );
  ```

### `src/internal/SwitchService.ts`
- Update `SwitchService` interface: add `create: boolean` to `run` input
- Add readline text prompt helper:
  ```ts
  const promptForBranchName = Effect.tryPromise({
    try: async () => {
      const rl = createInterface({ input: process.stdin, output: process.stdout });
      try { return (await rl.question("Branch name: ")).trim(); }
      finally { rl.close(); }
    },
    catch: (e) => new UnknownError(e, "Failed to read branch name"),
  });
  ```
- In `SwitchServiceImpl`, inject `GitService`
- Add create branch flow in `run` when `input.create === true`:
  1. Resolve repository (same picker logic as normal flow)
  2. `yield* ensureInteractive(promptForBranchName)` for branch name
  3. `yield* WorkTreeService.create({ repository, branch })` — creates the worktree
  4. `yield* tmux.attachSession(makeSessionName(repository, branch), resolveTargetPath(repository, branch))`
- Import `* as WorkTreeService from "@/Sandbox/adapter/WorkTreeService"` and `GitService`

## Key decisions
- `--branch` flag is ignored when `--create` is set (text input always used)
- `targetExists` check skipped in create path (worktree just created)
- Text input uses same readline pattern as `AgentCommandService.ts`

## Verification
```bash
# Normal switch (unchanged)
bun src/Cli.ts switch

# Create mode: should prompt for repo (picker), then text input for branch
bun src/Cli.ts switch --create

# Create with repo pre-specified: skip repo picker, prompt branch name only
bun src/Cli.ts switch --create --repository myrepo
```
